# 📖 How To Use This Wiki

---

## What This Wiki Is For

This wiki is a **quick-reference catalog** — a place to browse, discover, and get a fast summary of each visual style and resource repo. The full documentation, prompt packs, asset files, shaders, and deep dives all live in each style's **own dedicated repo**.

Think of it like a card catalog at a library: come here to find what you're looking for, then follow the link to the full book.

---

## How To Enable The GitHub Wiki Feature

The `wiki-pages/` folder in this repo contains ready-to-use markdown drafts. To turn on the actual GitHub Wiki tab:

1. Go to your repo on GitHub
2. Click **Settings**
3. Scroll to the **Features** section
4. Check the box next to **Wikis**
5. The **Wiki** tab will appear at the top of your repo

---

## How To Create Wiki Pages From These Drafts

Once the Wiki feature is enabled:

1. Click the **Wiki** tab on the repo
2. Click **New Page**
3. Copy the content from the corresponding file in `wiki-pages/` and paste it in
4. Set the page title to match the filename (e.g., `New-Age-Glam-Fantasy`)
5. Save

Repeat for each page. Start with `Home.md` — it becomes your wiki's landing page.

---

## How To Add A New Style Page

1. Copy `wiki-pages/_Page-Template.md`
2. Rename it to match your style (e.g., `wiki-pages/My-New-Style.md`)
3. Fill in all the sections
4. Add a row for it in `wiki-pages/Home.md`
5. Add a row for it in `README.md`

---

## A Note On GitHub Wikis

GitHub wikis are secretly **git repos themselves** — every wiki has its own clone URL. This means you can:

- Clone the wiki to your local machine
- Edit pages in your text editor
- Push changes back like any other repo

The wiki clone URL is always: `https://github.com/USERNAME/REPONAME.wiki.git`

---

## The Full Docs Always Live In Each Style's Own Repo

This wiki is for quick reference. The full deep-dive documentation — every prompt, every pillar explained in depth, every asset file — lives in the individual style repos. Follow the 🔗 links on each wiki page to get there.
