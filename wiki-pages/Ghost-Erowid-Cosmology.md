# 👻 Ghost Erowid Cosmology

> **What it is:** A psychedelic cosmology mapped as an operating system — a comprehensive system of motifs, entities, geometries, thresholds, emotions, and messages derived from Erowid trip reports, organized for art generation, shader development, and generative art.

🔗 **Full repo:** [merrypranxter/ghost-erowid-cosmology](https://github.com/merrypranxter/ghost-erowid-cosmology)

---

## Motif Type System

| Prefix | Type | Description |
|--------|------|-------------|
| `ENT_` | Entity | Beings and archetypes encountered |
| `LOC_` | Location | Spaces and OS states |
| `GEO_` | Geometry | Patterns, substrates, visual overlays |
| `THR_` | Threshold | Transitions and mode-switch events |
| `MSG_` | Message | Instructions and system downloads |
| `EMO_` | Emotion | Color grades and affective states |

---

## What's Inside

- **Canon motif definitions** — YAML + CSV files for every motif in the system
- **GLSL shaders** — one per concept, ready to drop into shader tools
- **Art direction files** — palettes, textures, motion behaviors, composition templates, entity codex
- **Prompt packs** — pre-built prompts, vocabulary hoards, prompt matrices

---

## Quick Start

1. Pick a motif (e.g. `GEO_INTERFERENCE_LATTICE` or `ENT_THE_ARCHIVIST`)
2. Look up its art hooks in the art direction files
3. Get the hex colors from the palette files
4. Assemble a prompt — or drop the shader directly into your tool
