# ✨ Astral

> **What it is:** A shader and art asset repository — GLSL/HLSL shaders, textures, palettes, AI art prompts and outputs, style references.

🔗 **Full repo:** [merrypranxter/astral](https://github.com/merrypranxter/astral)

---

## Repo Structure

| Folder | Contents |
|--------|----------|
| `shaders/` | Vertex, fragment, and compute shaders + examples |
| `assets/` | Textures, images, references, palettes |
| `ai-art/` | Prompts, outputs, styles, models |
| `scripts/` | Processing and conversion utilities |

---

## How Apps Discover Files

Apps find assets via `manifest.json` at the repo root — point your app at the repo and it can enumerate everything available.

## Naming Conventions

See the repo README for the full naming conventions table — files follow a consistent prefix/category system so tools can parse them automatically.
