# 🎨 style-wiki

> **Central catalog and quick-reference wiki for merrypranxter's visual style system repositories.**

This repo is a **browsable index and quick-reference catalog** — not a replacement for the full documentation. Each style lives in its own dedicated repo with deep docs, prompt packs, asset files, and everything else. Come here to browse, discover, and grab a quick summary before diving into a specific style.

## How to Use

- **Browse here** for quick summaries, keywords, and style pillars
- **Follow repo links** to get the full documentation, prompt packs, asset files, and deep dives
- **Enable the GitHub Wiki** on this repo (Settings → Features → Wikis) and copy pages from `wiki-pages/` to get a live browsable wiki
- **Add a new style** by copying `wiki-pages/_Page-Template.md` and filling it in

---

> The full deep-dive documentation always lives in each style's own repo. This wiki is for browsing, discovery, and quick reference.
